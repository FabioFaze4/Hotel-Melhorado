# Hotel

# Aviso
Projeto elaborado em Java
Executável em Terminal

# Opções
- Reservar Quarto
- Cancelar Reserva
- Listar Reservas
- Consultar Hóspede
- Editar Hóspede
- Registrar Consumo do Frigobar
- Check-out (Encerramento da Hospedagem)
